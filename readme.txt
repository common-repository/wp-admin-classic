Classic WordPress Admin theme (prior 3.8 style).

== Description ==

Recently upgraded to WordPress 3.8+ and not yet ready for the visual change? WP Admin Classic plugin simply disables new admin theme and reenables the prior 3.8 looks to your admin panel. Please, note that there is still some unsolved issues (see "Issue list), so if you have ideas how to solve it just shout!

Become our [**Facebook**](http://www.facebook.com/MindoMobileSolutions) fan to keep up with the latest updates. You can also find us on **Google+** circles [here](https://plus.google.com/116940685804905541644).

**Issue list:**

1. RTL Languages are not yet supported!!!

2. Post edit form turns blue (as a result for fixing http://1?ver=3.8 issue)

3. Admin bar is not displayed properly on front-end with standard Twenty Thirteen, Twenty Fourteen (updated themes).

**Notes:**

1. Dashboard widgets are now 100% wide. CSS fix by Robert Heller (Single column only!).

Please, report any other CSS/JS issues and suggestions how to fix them [here](http://wordpress.org/support/plugin/wp-admin-classic) or [here](http://mindomobile.com/contact-us/). Your help much appreatiated.

ENJOY!

== Installation ==

**There are 2 ways to install wp-admin classic on your blog:**

Upload the plugin to your blog plugin directory using ftp client or web file manager and activate it or;

Visit your blog admin panel and:

1. Click Plugins in the menu
2. Click Add New
3. In search field type "wp-admin classic" (without quotes)
4. Find wp-admin classic in the list and click Install Now.

== Upgrade Notice ==

To upgrade plugin, please copy and replace all existing files in wpadmin-classic directory. All necessary adjustments by the plugin will be done automatically. You can also update plugin using standard WordPress Update Manager!
